# **Ejercicio 6: Manejo de Ventanas y Pestañas Emergentes**

## **Descripción**

En este ejercicio aprenderás a manejar ventanas y pestañas emergentes en Selenium. Algunas páginas web abren nuevas ventanas o pestañas cuando interactúas con ciertos elementos. Aprenderás a cambiar el enfoque entre ellas y automatizar acciones en las ventanas o pestañas abiertas.

## **Tareas a Realizar**

1. Abre una página con múltiples enlaces que abran nuevas ventanas o pestañas.
2. Haz clic en un enlace para abrir una nueva ventana/pestaña.
3. Cambia el foco a la nueva ventana/pestaña y realiza alguna acción (como verificar el título o contenido).
4. Regresa a la ventana/pestaña original y verifica que se mantenga la integridad de la información.

## **Requisitos**

- **Selenium WebDriver** para Java o JavaScript.
- **ChromeDriver** (u otro WebDriver compatible).
